# C# JSON and XML Deserialization Project

## Features
- Manual JSON file creation
- XML Reader example
- JSON deserialization using Newtonsoft.Json
- Console output with loops
- This work is divided into mainly three task, first making XML file manually and then making XML Reader file to print output of users. Then making Json file manually where i store the users name, age and city, and then deserialize it with the help og loop. It gives the output of users with there features.
- On the next part i created a Json file where i store Admins name, age, city and role. Then upon deserilizing it with the help of loop, it also gives the admins output

## Technologies
- C#
- .NET
- Newtonsoft.Json
- XML


## How to Run
1. Open project in VS Code
2. Run:
   dotnet run
3. First run only XML contain. In program.cs file i inherit the XML reader file.
4. Second, run only JSON first part without admin, and then the JSON with Admin
